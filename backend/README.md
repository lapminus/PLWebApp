## Premier League Backend (PL)

A Spring Boot REST API for managing football (soccer) player data.
This project focuses on the backend, following clean REST design, JPA persistence, validation, and error handling.

## Features

- CRUD operations for players
- PostgreSQL database integration
- Docker for containment
- Spring Data JPA repositories
- Validation with custom error responses
- Global exception handling
- Transactional updates

### Data
The dataset used in this project was obtained from an external source.
